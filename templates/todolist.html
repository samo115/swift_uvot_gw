UVOT Gravitational Wave followup checklist.

Checking thumbnails

  • Keep and eye on GCNs
  • Go to MSSL gwaves thumbnail page. Sort by probability to check sources with high probability first, then sort by type to then check Q0 objects, followed by everything else. Push interesting candidates to the Swift TOM page (will be a button on the gwaves entry)
  • Keep an eye out for XRT sources. From Leicester GW page page click on XRT link for GW event, then for a particular source use the 'observation list' to find obsids to download.
  • For other sources, you can find the OBSID for a given coordinate by using the GW-XRT page "Fields Planned - search" tool.

Further Candidate source checks

  • go to CDS Portal and put in the coordinate to get an easy way to view the position in lots of images.
  • go to Vizier and put the coordinate in the "search by position", modify "target dimension" to 5 arcsec! Click on Go! (not 'find catalogs') Check sources found especially GAIA DR2 (g band),WISE (IR), AAVSO
  • Option: construct an SED from the catalogs using Vizier SED catalogs - this can give an idea of the sort of precursor or absence of it.
  • Is this a minor planet?
  • or manmade satellite?
  • Is the source near the edge which shows shift-and-add? Check if there is a star on/near(<20 arcsec) the opposite edge (compare sky image to larger DSS image and blink; use line region parallel to edge to line up)
  • is the source fuzzy or small and there is a very bright source nearly mirrored through the centre of the detector (it is not precisely mirrored but within ~200 pixels)
  • add new transient to TNS

Updating the world

  • Send GCN out for good candidates (Draft GCN), leave comment on TOM that GCN has been submitted, with the GCN number.
  • Send final summary GCN when all thumbnails have been checked (Draft GCN).

Useful links

  • Leicester GW page
  • Franks thumbnail page
  • Swift LVK wiki page
