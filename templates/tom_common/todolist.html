UVOT Gravity Wave followup checklist.

Keeping track of data

    read relevant GCNs
  • load XRT-GW page https://www.swift.ac.uk/GW/ and follow links for the latest GW to 'Plan' and 'XRT'. Take down info on distance, type of object etc.
  • From XRT-GW page follow link to 'UVOT analysis' and note the number in the web address: this should be the number containing the new data (needed for next two bullets).
  • Check Frank's thumbnail page (https://swift.gsfc.nasa.gov/uvot_tdrss/thumbnails/) to find latest - use sort by time.
  • Go to MSSL's thumbnail page (https://www.mssl.ucl.ac.uk/gwave/prot/). Sort by type to check Q0 objects first.
  • Keep an eye out for XRT sources. From XRT-GW page click on a particular source and use the 'observation list' for obsids to download.
  • For other sources, you can find the OBS for a given coordinate by using the GW-XRT page "Fields Planned - search" tool.

Updating

  • If not already made, make a new page for whole Swift team to check out sources by going to team page wiki, scrolling down to bottom, clicking on GWTriggers (GWTriggers) and following instructions to add new trigger e.g. S190728q.
  • Make a new page for UVOT only, by going to GWTriggers page (GWTriggers) and repeating with e.g.GW190728q, but this time don't fill in the table.
  • Fill in table instead in UVOTGWTriggers (UVOTGWTriggers) page (also found at by navigating from bottom of main team page)
  • Record any new sources of interest you find on the UVOTGWTriggers page for other UVOT team members to comment and check.
  • Any that are deemed interesting enough to follow up with ToOs and GCNs should be recorded on the GWTriggers pages.

Further Candidate source checks

  • go to CDS Portal and put in the coordinate to get an easy way to view the position in lots of images.
  • go to Vizier and put the coordinate in the "search by position", modify "target dimension" to 5 arcsec! Click on Go! (not 'find catalogs') Check sources found especially GAIA DR2 (g band),WISE (IR), AAVSO
  • Option: construct an SED from the catalogs using Vizier SED catalogs - this can give an idea of the sort of precursor or absence of it.
  • Is this a minor planet?
  • or manmade satellite?
  • Is the source near the edge which shows shift-and-add? Check if there is a star on/near(<20 arcsec) the opposite edge (compare sky image to larger DSS image and blink; use line region parallel to edge to line up)
  • is the source fuzzy or small and there is a very bright source nearly mirrored through the centre of the detector (it is not precisely mirrored but within ~200 pixels)
  • add new transient to https://wis-tns.weizmann.ac.il/ (TBD details)
